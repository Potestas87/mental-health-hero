# V0.2 Alpha Change Notes

## Overview
This document summarizes what was added/updated for the `V0.2 Alpha` milestone and the intent behind each change.

## Core Additions
1. Splash flow added (`SplashScene` + `SplashVideoController`)
- Added a dedicated splash scene to show studio logo video before gameplay routing.
- Added fade-out transition into `BootstrapScene` for cleaner startup UX.
- Updated input handling in splash logic to use the new Input System safely.

2. Scene music system added (`HomeMusicPlayer`)
- Added looped background music manager that persists across scene loads.
- Configured exclusions so music does not play in:
  - `DungeonScene`
  - `OnboardingScene`
  - `SplashScene`
  - `BootstrapScene`

3. Scene backgrounds added
- Added static background images for key non-dungeon UI scenes.
- Wired scene-level backgrounds for Home/Task/Upgrade/Cosmetics/Pomodoro screens.

## Animation/Visual Fixes
1. Warrior controller transition fix
- Patched `Warrior.controller` transition durations to remove one-frame facing snap when movement ends.
- This aligns behavior with prior Mage/Ranger transition fixes.

2. Enemy visual polish
- Added enemy damage red flash feedback in `EnemyController2D`.
- Added directional animation-driving parameters in enemy runtime behavior.

3. Enemy + boss animation pipeline support
- Added editor automation script to build directional clips/controllers and assign enemy prefab animators.
- Included boss archetype in the same directional animation workflow.

## Cosmetics/Data Updates
1. Category-ready cosmetics model
- Updated cosmetics data flow toward category-based equip structure.
- Kept MVP active category as tint-only while preserving future extensibility.

2. Default/no-tint support
- Added `tint_default` (no-tint) option.
- Added default unlock behavior so baseline tint options are available for testing.

## Build/Project Config Updates
1. Build settings updated
- Added splash scene to startup flow and preserved scene routing order.

2. Asset imports for alpha content
- Added splash video and background music runtime assets.
- Added scene background art assets used by UI scenes.

## Notes for Reviewers/Testers
- This alpha focuses on validating the complete loop and startup UX polish, not final balancing or full content polish.
- iOS device build/distribution remains intentionally deferred for a later milestone.
